# ♟️ Shadowchess

**Shadowchess** is a high-speed, fair, and competitive open-source chess platform. Built for performance and precision.

From **Vietnam** 🇻🇳 with passion.

## 🚀 Core Features
- **0s Premove:** Smooth and ultra-fast gameplay with zero-second premoves (Lichess style).
- **Anti-Cheat System:** Integrated detection with a **3-Strike Rule** to maintain a fair environment.
- **Real-time Matchmaking:** Instant and stable pairing using Socket.io.
- **Global Leaderboard:** Track rankings across Bullet, Blitz, and Rapid categories.
- **Modern UI:** Minimalist dark-themed interface designed for focus.

## 🛠️ Tech Stack
- **Backend:** Node.js, Express
- **Real-time:** Socket.io
- **Database:** MongoDB
- **Chess Engine/Logic:** Chess.js & Chessboard.js
